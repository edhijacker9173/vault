# The King Shelf

Static Stephen King bibliography explorer based on the original Lovecraft site structure, redesigned as a dark paperback catalog interface.

Open `index.html` directly in a browser. The bibliography data lives in `data.js` as `window.STORIES`; the local hero image lives at `assets/king-shelf-hero.png`.

Scope: released novels, Bachman novels, novellas, story collections, nonfiction, and major coauthored books. Source dates/categories were checked against Stephen King's official Written Works page: https://stephenking.com/works/
