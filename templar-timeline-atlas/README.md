# Templar Timeline Atlas

Static interactive web app built from `Downloads/Files_by_Type/MD/knights_templar_briefing.md`.

Open `index.html` directly in a browser. The app has no build step and no external JavaScript dependencies.

Files:
- `index.html` - page shell and controls
- `styles.css` - visual design and responsive layout
- `data.js` - structured events, categories, and approximate map locations
- `app.js` - timeline, filters, map pins, routes, and event detail behavior
- `assets/templar-map-hero.png` - generated local hero asset

Scope: the timeline emphasizes the First Crusade, the Templars' founding and recognition, their military and financial rise, the fall of Acre, the 1307 arrests, dissolution, Jacques de Molay's execution, and the later treasure/myth layer.
